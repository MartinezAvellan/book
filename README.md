### POST - /book/create-book-review-and-rating
#### REQUEST
```json
{
    "book_id": 5,
    "rating": 2,
    "review": "Este livro eh medio 2"
}
```
#### RESPONSE: 200
```json
{
    "id": 1,
    "book_id": 5,
    "rating": 2,
    "review": "Este livro eh medio 2"
}
```
